Dies ist eine Option von WIRIS Quizzes für ausgearbeitete Mathematik- und Wissenschaftsfragen.

Dieses Datenfeld ist fakultativ. Wenn es benutzt wird, muss eine boolesche Variable oder ein numerischer Wert zwischen 0 und 1 eingegeben werden. Der Wert dieser Variable wird die zugewiesene Note für diese Auswahl sein.

Im WIRIS Quizzes Benutzerhandbuch finden Sie weitere Informationen.
