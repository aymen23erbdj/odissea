Dies ist eine Option von WIRIS Quizzes für ausgearbeitete Mathematik- und Wissenschaftsfragen.

Um dieses Feld zu aktivieren, muss der WIRIS Editor geöffnet sein. Wenn sie aktiviert wird, erscheint im Antwortfeld eine WIRIS CAS Session, wo der Schüler Hilfsrechnungen durchführen kann.

Im WIRIS Quizzes Benutzerhandbuch finden Sie weitere Informationen.
