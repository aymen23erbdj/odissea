Dies ist eine Option von WIRIS Quizzes für anspruchsvolle Mathematik- und Wissenschaftsfragen.

Wenn sie aktiviert wird, öffnet sich im Antwortfeld eine WIRIS CAS Session, wo Hilfsrechnungen durchgeführt werden können. Für die Fragesorte Freitext (Essay), kann die WIRIS CAS Session den HTML Editor ersetzen, sodass der Schüler seine Antwort im WIRIS CAS eingeben muss. Wenn die Fragesorte Kurzantwort aktiviert ist, und wenn das Feld WIRIS Editor aktiviert ist, können mithilfe der daraufhin erscheinenden Schaltfläche die Ergebnisse in den WIRIS Editor übertragen werden.

Im WIRIS Quizzes Benutzerhandbuch finden Sie weitere Informationen.
