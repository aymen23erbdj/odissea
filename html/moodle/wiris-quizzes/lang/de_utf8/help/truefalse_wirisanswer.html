Dies ist eine Option von WIRIS Quizzes für ausgearbeitete Mathematik- und Wissenschaftsfragen.

Dieses Datenfeld ist fakultativ. Wenn benutzt, ist sein Wert ein Variablenname. Diese Variable muss boolesch sein (ihr Wert musst echt oder falsch sein). Üblicherweise ist es das Ergebnis der Auswertung einer Subjunktion. Die richtige Antwort ist demenstprechend der Wert dieser Variable.

Im WIRIS Quizzes Benutzerhandbuch finden Sie weitere Informationen.
