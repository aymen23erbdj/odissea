Ceci est une option de WIRIS Quizzes qui fournit des questions perfectionnées en maths et en sciences.

Si elle est activée, le formulaire de réponse affichera un applet WIRIS CAS où l’élève pourra effectuer des calculs auxiliaires. Dans le cas des questions de type Réponse ouverte, il est possible d’indiquer que l’applet remplacera l’éditeur HTML, pour que l’étudiant réponde directement dans WIRIS CAS. Dans les questions de type Réponse courte et aussi si l’option Editeur de formules a été sélectionnée, une touche s’affichera pour copier le résultat des calculs effectués dans l’éditeur de formules.

Consulter le manuel de WIRIS Quizzes pour davantage de renseignements.
