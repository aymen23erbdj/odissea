GeoGebra activity file

The .ggb file with the GeoGebra activity.

  • To add a GeoGebra file to the course you have to click the Choose or upload a file... button.
  • A new window will be opened, which allows you uploading files and/or selecting some of the existing ones. You have to click on the Choose link to select the desired file.


IMPORTANT: A variable named grade must be defined in each GeoGebra activity to allow Moodle save properly the score, which will be updated depending on the user actions. If it doesn't exist, the grade will be always 0. Therefore, if the grade isn't defined, it's recommended to select No grading option as Grading Method.
