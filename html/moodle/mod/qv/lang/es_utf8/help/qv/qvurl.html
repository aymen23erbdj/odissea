Enlace de un Cuaderno

El enlace es la dirección del fichero index.html del Cuaderno que muestra, en la parte central, el número y nombre de las hojas que contiene este Cuaderno.


Pantalla inicial de un Cuaderno: index.html

Hay formas diversas de obtener el enlace del Cuaderno:

  • Para añadir el enlace de un Cuaderno de la Biblioteca, hay que copiar la dirección que aparece en su ficha.


    Enlace de un Cuaderno de la Biblioteca: http://clic.xtec.cat/qv_viewer/quaderns/biblioteca/clima_ca

  • Si se desea añadir un Cuaderno propio creado con el Editor de Cuadernos Virtuales, hay que copiar la dirección que aparece en la parte superior de la pantalla de edición del Cuaderno.


    Enlace de un Cuaderno del Editor: http://clic.edu365.cat/qv_viewer/quaderns/sarjona/planetes/

  • Para añadir un Cuaderno a los ficheros del curso se tiene que utilizar el botón Elija o suba un archivo ... Cuando se clica este botón, se abre una ventana nueva que permite subir archivos nuevos y/o seleccionar los que ya contiene el curso. Para seleccionar uno de estos ficheros, se ha de clicar sobre el enlace Elegir que aparece al lado. Esta página contiene información detallada de cómo subir, descomprimir y elegir un Cuaderno a un curso Moodle.


    Enlace al fichero index.htm del Cuaderno dentro de un curso Moodle: qv/introqv/html/index.htm


  • Además, también se pueden añadir enlaces a Cuadernos Virtuales publicados en otros servidores web, como por ejemplo:

    http://www.miservidor.com/carpeta/cuaderno/html/index.htm
