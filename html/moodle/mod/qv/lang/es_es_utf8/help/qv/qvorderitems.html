Ordenación aleatoria de hojas y preguntas

Si se activa esta opción, el orden en que se mostraran las hojas y preguntas del Cuaderno que lo permitan será diferente para cada alumno/a.
Por lo tanto, para que algunas o todas las hojas y preguntas del Cuaderno aparezcan con un orden diferente para cada alumno/a hay que editar el Cuaderno Virtual con el Editor e indicar las hojas y/o preguntas que pueden aparecer de forma aleatoria.
Para conseguirlo se ha añadido el parámetro Orden a las pantallas de edición de las hojas y preguntas del Editor que permite indicar, una a una, las hojas y preguntas que pueden aparecer en cualquier lugar del Cuaderno (seleccionando la opción La hoja/pregunta se puede mostrar en CUALQUIER POSICIÓN dentro del Cuaderno).
