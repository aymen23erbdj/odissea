Barra de navegació

Esta opción añade una barra en la parte superior de la visualización de los cursos que ofrece opciones de navegación adicionales como acceder de forma rápida a las calificaciones o a otros temas / semanas del curso. Hay que tener en cuenta que esta barra aparecerá tanto en la visualización de profesorado como en la de alumnado.

  • Barra de navegación avanzada.
    • Mostrar la barra de navegación avanzada con autoselección de tema. Además de la barra de navegación, al alumnado se le mostrará sólo el tema 0 y el tema / semana marcado como actual. En caso de que no haya ninguno marcado sólo se mostrará el tema 0. Este comportamiento varía ligeramente con el formato "Sencillo por temas", ya que cuando hay un tema marcado como actual no se mostrará el tema 0.
    • Mostrar la barra de navegación avanzada sin autoselección de tema. Además de la barra de navegación, al alumnado se le mostrará sólo el tema 0 y el tema / semana marcado como actual. Si no hay ninguno marcado se mostrarán todos los temas / semanas. Si se usa el formato "Sencillo por temas" y no hay ningún tema marcado como actual, se mostrará sólo el tema 0.
    • No mostrar la barra de navegación avanzada. The bar will not be shown and the course will have the usual behavior.

  • Mostrar enlace a cualificaciones. En la barra de navegación aparecerá un enlace con el icono () para acceder directamente al apartado de calificaciones. Esta opción sólo se puede activar cuando la barra de navegación esté activa.

  • Mostrar enlace para ver todos los temas. En la barra de navegación aparecerá un enlace que permite mostrar todos los temas / semanas, estén o no marcados como actual. Esta opción sólo se puede activar cuando la barra de navegación esté activa.

  • Mostrar una lista de enlaces a cada tema. En la barra de navegación aparecerá una lista numerada donde cada elemento permitirá ir a un tema / semana, esté o no marcado como actual. Esta opción sólo se puede activar cuando la barra de navegación esté activa.

  • Mostrar bloques laterales (sobrescribe los parámetros de configuración del tema). Permite elegir si se quiere que aparezcan los bloques laterales en la visualización del alumnado.

Recordad que para marcar un tema/semana como actual hay que activar la edición i seleccionar el icono .
