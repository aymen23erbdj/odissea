Iconos para el formato Sencillo

El formato "Sencillo por temas" afecta a la visualización del curso de los usuarios con rol estudiante de forma que todos los recursos y actividades se muestran con un icono, que tiene un tamaño mayor de lo habitual y, incluso, se puede personalizar. Este formato lleva un conjunto de iconos que no varían aunque se seleccione otra apariencia y / o juego de iconos

  • Usar la imagen existente. Permite mantener el icono actual para el recurso/actividad, usar el icono predeterminado del formato Sencillo para cada uno de los tipos de recursos y actividades, o usar una de los iconos predefinidos para diferentes áreas.

  • Subir una imagen. Permite elegir una imagen para personalizar el icono que debe representar al recurso / actividad. La imagen debe estar en formato JPG, PNG o GIF y es recomendable que sea de 128x128 píxeles (es decir, 128px de ancho y 128px de altura). En caso de que el tamaño sea diferente, la imagen posiblemente se verá deformada o con mala calidad.
