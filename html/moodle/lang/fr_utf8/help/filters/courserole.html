Filtre rôle du cours

Ce filtre vous permet de sélectionner des utilisateurs sur la base du rôle qui leur est attribué dans un cours spécifié par son nom abrégé ou dans une catégorie de cours. Si le champ Nom abrégé est vide, la catégorie est « Toutes les catégories » et le rôle est « Tous les rôles », le filtre est n'est pas actif.
