Ajouter des sujets

Vous pouvez utiliser ce lien pour ajouter un sujet à un dialogue. C'est une bonne idée qu'un dialogue ait toujours un sujet, cela permet de rester dans le cadre et de focaliser les réponses sur celui-ci. Si vous voulez commencer à parler d'un autre sujet il est conseillé de fermer le dialogue courant et d'en démarrer un autre sur le nouveau.
