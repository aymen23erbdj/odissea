Navigation bar

This option adds a toolbar at the top of the display of the courses offering as additional navigation options to quickly access the grades or other issues / weeks of the course. Keep in mind that this bar will appear for both teachers and students.

  • Advanced Navigation Bar.
    • Show the advanced navigation bar and autoselect the topic. In addition to the navigation bar, the students will be shown only topic 0 and the topic / week marked as current. If there is any item marked only the topic 0 will be shown. This behavior varies slightly with the format "Simple Topics format", as when an item is marked as current does not show the topic 0.
    • Show the advanced navigation bar but not autoselect the topic. In addition to the navigation bar, students will be shown only topic 0 topic / week flagged as current. If there is any marked will be displayed all the topics / weeks. If you use the format "Simple themes" and no topic is marked, only topic 0 will be shown.
    • Don't show the advanced navigation bar. The bar will not be shown and the course will have the usual behavior.

  • Show link to grades. In the navigation bar will appear a link with an icon () to go directly to the section grades. This option can be activated only when the navigation bar is active.

  • Show a link to see all topics. In the navigation bar will appear a link to show all topics / weeks, whether or not marked as present. This option can be activated only when the navigation bar is active.

  • Show a list of links to each topic. In the navigation bar will appear a numbered list where each item will go to a topic / week, whether or not marked as current. This option can be activated only when the navigation bar is active.

  • Show side blocks (overrides the theme settings). Allows you to choose if you want to show the side blocks in the students view.

Remember that to mark a topic / week as current, you have to enable editing and select the icon.
