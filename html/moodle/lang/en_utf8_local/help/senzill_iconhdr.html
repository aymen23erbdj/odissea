Icon for Senzill format

The format "Simple Topics format" affects the view of the course for student so that all resources and activities are displayed with an icon, which is larger than usual and even can be customized. This format has a set of icons that are larger than those shown regularly and therefore do not change even if you select a different appearance and / or set of icons.

  • Use existing image. Allows to maintain the current icon for the resource / activity, use the default icon for each of the types of resources and activities, or use one of the predefined icons for different areas.

  • Upload an image. Allows you to choose an image to customize the icon that represents the resource / activity. The image must be in JPG, PNG or GIF and the recommended size is 128x128 pixels (128px wide and 128px high). If the size is higher / lower, the image will possibly warped or poor quality.
